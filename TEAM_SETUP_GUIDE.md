## Quick Start for Team Members

### 1. Clone the Repository
```bash
git clone https://github.com/Cirbble/AURAlos.git
cd AURAlos/auralos
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```

Then open: **http://localhost:5173**

### 4. Build for Production
```bash
npm run build
```

## What's Working

### Refine Search Feature
1. Navigate to "AI Discover" tab
2. Search for products or upload image
3. View top 3 results
4. Click **"Refine Search"**
5. Answer **3 Likert questions**:
   - Overall satisfaction
   - Style match
   - Price satisfaction
6. Add optional text feedback
7. Get refined results
